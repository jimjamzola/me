Semina problems and pedagogy

I use semina-style problem scaffolding in advanced courses to encourage structured problem solving and incremental discovery. The semina format emphasizes self-contained problems with a short instructional scaffolding block. For an implementation and examples, see Semina.

Semina was developed by Zilin Jiang. For information about the project and origin, see Semina (Zilin Jiang, creator).

Example semina-style resource (coming soon): Semina problem set examples

Teaching philosophy

I prioritize active learning, careful problem sequencing, and connecting numerical tools to rigorous analysis. I use semina-style problems as an auxiliary tutoring resource to build proof fluency and computational intuition.
